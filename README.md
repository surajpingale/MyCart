# MyCart

MyCart allows to user buy and sell their products online.


## Technologies

* Kotlin
* Firebase Firestore
* Glide
* Recycler View
* MVVM
* Coroutines
* Live Data
* Navigation Component
* Diff Utils
* Custom View
